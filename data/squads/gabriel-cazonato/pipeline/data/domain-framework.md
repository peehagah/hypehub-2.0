# Domain Framework: Influencer Hub

Framework operacional para produção integrada de conteúdo + tráfego pago para criadores e influenciadores digitais.

## Framework Central: PIPELINE DE CONTEÚDO INFLUENCER (PCI)

### Fase 1: Descoberta e Contexto
1. **Briefing do cliente** — Identificar o influenciador, nicho, tema da rodada e objetivo de negócio (awareness, leads, vendas)
2. **Análise de audiência** — Perfil demográfico e psicográfico do público do criador
3. **Pesquisa de tendências** — Identificar o que está em alta no nicho (trends, hashtags, formatos)
4. **Benchmark de concorrentes** — Analisar o que os pares do criador estão publicando com sucesso

### Fase 2: Ideação e Ângulo
1. **Seleção de pauta** — Escolher o tema principal com base em tendências + audiência + objetivo
2. **Geração de ângulos** — Criar 5 perspectivas emocionais para a mesma pauta (medo, oportunidade, educacional, contrário, inspiracional)
3. **Seleção de ângulo** — Escolher o ângulo com maior potencial para o momento e objetivo
4. **Hook principal** — Definir o hook do conteúdo (frase/visual de abertura) antes de escrever qualquer coisa

### Fase 3: Produção de Conteúdo
**Instagram Carrossel (8–10 slides):**
1. Slide 1: Capa com hook visual + título ≤7 palavras
2. Slide 2: Agravamento do problema/curiosidade
3. Slides 3–8: Um conceito por slide, hierarquia visual clara (40–80 palavras/slide)
4. Slide 9: Resumo/lista com "Salva para não perder"
5. Slide 10: CTA específico e contextualizado
6. Legenda: 5 partes (hook acima do fold → desenvolvimento → lista → CTA → hashtags)

**Instagram Reels (15–30s ideal):**
1. Hook (0–2s): texto na tela + narração que interrompe o scroll
2. Setup (2–5s): contexto rápido do que vem a seguir
3. Delivery (5–25s): valor com cuts visuais a cada 3–5 segundos
4. CTA (últimos 3–5s): pedir ação específica
5. Design de loop: final conecta visualmente ao início para incentivar re-assistência

**YouTube Script:**
1. Hook (0–30s): entregar spoiler do melhor momento + estabelecer stakes
2. Intro (30s–1min): por que este tópico importa, brevemente
3. Body (1min–80%): 3–5 seções com open loops + pattern interrupts a cada 30–60s
4. Climax (60–70% do vídeo): insight principal ou "aha moment"
5. Summary (últimos 1–2min): recap + CTA para próximo vídeo

### Fase 4: Controle de Qualidade
1. **Revisão do criativo** — Hook está nos primeiros 3 segundos? CTA é específico? Formato certo?
2. **Alinhamento com pilares** — O conteúdo se encaixa em um dos pilares (Hero/Hub/Help)?
3. **Benchmark de qualidade** — Comparar com referências de alta performance do nicho
4. **Score de aprovação** — Verificar critérios antes de aprovar

### Fase 5: Tráfego Pago
1. **Identificar vencedores orgânicos** — Conteúdo com ER >2x a média do criador é candidato a boost
2. **Definir funil** — Topo (awareness), meio (consideração) ou fundo (conversão)?
3. **Configurar estrutura de campanha** — CBO para escala, ABO para testes
4. **Definir criativos para teste** — 3 hooks, 2 formatos, 2 CTAs
5. **Definir KPIs e thresholds** — CPM, CTR, Hook Rate para validação

## Decisões Estratégicas

- **Quando usar Carrossel vs Reel:** Carrossel → conteúdo educativo/explicativo (alta taxa de salvamento). Reel → conteúdo de descoberta/trending (alto alcance).
- **Quando impulsionar conteúdo orgânico:** Quando o ER orgânico for >2x a média do criador nas primeiras 24–48h.
- **Quando criar novo ad set vs duplicar:** Sempre duplicar e editar; nunca criar do zero (perde dados históricos).
- **Quando trocar criativo:** Quando o CTR cair >20% em relação ao início da campanha na mesma semana.
- **Quando escalar budget:** Aumentar no máximo 20–30% a cada 2–3 dias para não reiniciar a fase de aprendizado.
