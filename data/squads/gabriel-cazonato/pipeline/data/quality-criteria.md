# Quality Criteria: Influencer Hub

Critérios de avaliação de qualidade para todo conteúdo produzido pelo squad.
Cada critério é pontuado de 1–10. Aprovação exige média ≥7 e nenhum critério abaixo de 4.

---

## Critérios de Conteúdo Instagram

### Carrossel

| Critério | Peso | Descrição |
|---|---|---|
| Força do Hook | Alto | O Slide 1 para o scroll? Hook nos primeiros 3 segundos visuais? |
| Clareza por slide | Alto | Apenas 1 ideia por slide? 40–80 palavras? Hierarquia visual? |
| Progressão narrativa | Médio | Slides seguem lógica (problema → agravamento → solução → prova → CTA)? |
| CTA específico | Alto | CTA contextualizado (não genérico)? Pede ação específica? |
| Legenda estruturada | Médio | Hook acima do fold? Parágrafos curtos? Hashtags no final (3–5)? |
| Potencial de salvamento | Alto | Conteúdo tem valor de referência que justifica salvar? |
| Alinhamento com persona | Alto | O conteúdo fala com a audiência do influenciador especificamente? |

**Thresholds:**
- Hook < 4 = REJEIÇÃO IMEDIATA (hard reject)
- Média < 7 = REJEIÇÃO
- Média ≥ 7, algum critério entre 4–6 = APROVAÇÃO CONDICIONAL com ajustes

### Reels (Script)

| Critério | Peso | Descrição |
|---|---|---|
| Hook (0–2s) | Alto | Texto + narração que provocam curiosidade imediata? |
| Duração adequada | Médio | Entre 15–30s? |
| Loop design | Médio | Final conecta ao início para incentivar re-assistência? |
| Roteiro para produção | Alto | Script está claro o suficiente para o criador gravar? |
| CTA específico | Alto | CTA diferente de "curta e se inscreva"? |
| Legenda otimizada | Médio | Hook nos primeiros 125 caracteres? |

## Critérios de Conteúdo YouTube

| Critério | Peso | Descrição |
|---|---|---|
| Conceito de Thumbnail | Alto | Há um conceito visual claro com ≤4 palavras de texto? |
| Título otimizado | Alto | 50–70 caracteres? Keywords na frente? Curiosity gap + benefício? |
| Hook dos primeiros 30s | Alto | Entrega spoiler do payoff + estabelece stakes? |
| Estrutura em 3 atos | Médio | Open loops, climax a 60–70%, CTA no final? |
| Pattern interrupts | Médio | Há cues para B-roll/gráficos a cada 30–60s? |
| CTA final | Médio | Um CTA principal claro (não 5 pedidos simultâneos)? |

## Critérios de Tráfego Pago

| Critério | Peso | Descrição |
|---|---|---|
| Estrutura de funil | Alto | Campanha cobre topo + meio + fundo? |
| Tamanho de audience | Alto | Audience entre 200k–5M por ad set? |
| Variedade criativa | Médio | Mínimo 3 hooks diferentes? 2 formatos? |
| KPIs definidos | Alto | CPM, CTR, Hook Rate e CPA definidos por fase? |
| Thresholds de otimização | Médio | Critérios de escala, pausa e troca de criativo definidos? |

---

## Critérios Globais (aplicam a todo conteúdo)

- [ ] Conteúdo não copia formato idêntico de um concorrente direto do influenciador
- [ ] Tom de voz está alinhado com o estilo pessoal do criador (não parece AI-generated óbvio)
- [ ] Hashtags relevantes ao nicho (não genéricas como #instagood #love)
- [ ] Nenhum conteúdo viola diretrizes da plataforma (claims falsos, imagens enganosas)
- [ ] CTA é único e específico (não há dois CTAs conflitantes na mesma peça)

---

## Escala de Decisão do Revisor

| Pontuação | Decisão |
|---|---|
| Média ≥ 7/10, nenhum critério < 4 | ✅ APROVADO |
| Média ≥ 7/10, critério não-crítico entre 4–6 | ⚠️ APROVAÇÃO CONDICIONAL (ajustes menores) |
| Média < 7/10 | ❌ REJEITADO |
| Qualquer critério de peso ALTO < 4 | ❌ REJEITADO (hard reject) |
| 3+ rodadas de revisão do mesmo conteúdo | 🚨 ESCALAR para Pedro |
